Jump Scares

There are many ways to implement jump scares in MineCraft using command blocks.

One way to implement a jump scare with a command blocks is to have the blocks automatically play a specific sound you want whenever something specific happens.

A example of this would be having the sound of a creeper about to explode play, whenever a player mines diamond ore. This would cause the player to jump, thinking they are about to loose all the new found diamonds to the sound being played automatically.
